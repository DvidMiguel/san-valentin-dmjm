Python Cheat Sheet — Fundamentos

Python Base

1. Variables

Permiten almacenar valores.

nombre = "David"
edad = 29
altura = 1.81

2. Tipos de datos

int, float, str, bool

numero = 10
precio = 19.99
texto = "Hola"
activo = True

3. Operadores

Operaciones matemáticas y comparaciones.

a = 10
b = 3

suma = a + b
division = a / b
mayor = a > b

4. Condicionales

Ejecutan código dependiendo de condiciones.

edad = 18

if edad >= 18:
    print("Mayor de edad")
else:
    print("Menor de edad")

5. Listas

Colección ordenada de valores.

numeros = [1,2,3,4]
print(numeros[0])

6. Diccionarios

Estructura clave → valor.

persona = {
 "nombre": "David",
 "edad": 29
}

print(persona["nombre"])

7. Bucle for

Permite recorrer estructuras.

numeros = [1,2,3]

for n in numeros:
    print(n)

8. Bucle while

Se ejecuta mientras la condición sea verdadera.

contador = 0

while contador < 3:
    print(contador)
    contador += 1

9. Funciones

Permiten reutilizar código.

def sumar(a, b):
    return a + b

resultado = sumar(5,3)

10. Importación de módulos

Permite usar librerías externas.

import math

print(math.sqrt(16))

Listas - Métodos principales

Operaciones más usadas con listas.

lista = [1,2,3]

# Añadir
lista.append(4)
lista.insert(1, 10)
lista.extend([5,6])

# Eliminar
lista.pop()
lista.remove(10)
lista.clear()

# Consulta
len(lista)
lista.count(2)
lista.index(3)

# Orden
lista.sort()
lista.reverse()

Diccionarios - Operaciones

Creación, modificación y eliminación.

dic = {"nombre":"Juan","edad":25}

# Acceso
dic["nombre"]
dic.get("ciudad","NA")

# Añadir / modificar
dic["ciudad"] = "Madrid"
dic.update({"profesion":"Data"})

# Eliminar
dic.pop("edad")
del dic["ciudad"]
dic.clear()

Iteración Diccionarios

Recorrer claves y valores.

dic = {"a":1,"b":2}

for k in dic.keys():
    print(k)

for v in dic.values():
    print(v)

for k,v in dic.items():
    print(k,v)

List Comprehension

Crear listas usando for en una sola línea.

numeros = [1,2,3,4,5]

cuadrados = [n**2 for n in numeros]

pares = [n for n in numeros if n % 2 == 0]

Dictionary Comprehension

Crear diccionarios usando for.

numeros = [1,2,3,4]

dic = {n: n**2 for n in numeros}

pares = {n: n**2 for n in numeros if n%2==0}

For + estructuras (patrón base)

Base de transformación de datos.

resultado = []

for n in range(1,6):
    if n % 2 == 0:
        resultado.append(n**2)
