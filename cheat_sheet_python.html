Python Cheat Sheet — Fundamentos

Python Base

1. Variables

Permiten almacenar valores.

nombre = "David"
edad = 29
altura = 1.81

2. Tipos de datos

int, float, str, bool

numero = 10
precio = 19.99
texto = "Hola"
activo = True

3. Operadores

Operaciones matemáticas y comparaciones.

a = 10
b = 3

suma = a + b
division = a / b
mayor = a > b

4. Condicionales

Ejecutan código dependiendo de condiciones.

edad = 18

if edad >= 18:
    print("Mayor de edad")
else:
    print("Menor de edad")

5. Listas

Colección ordenada de valores.

numeros = [1,2,3,4]
print(numeros[0])

6. Diccionarios

Estructura clave → valor.

persona = {
 "nombre": "David",
 "edad": 29
}

print(persona["nombre"])

7. Bucle for

Permite recorrer estructuras.

numeros = [1,2,3]

for n in numeros:
    print(n)

8. Bucle while

Se ejecuta mientras la condición sea verdadera.

contador = 0

while contador < 3:
    print(contador)
    contador += 1

9. Funciones

Permiten reutilizar código.

def sumar(a, b):
    return a + b

resultado = sumar(5,3)

10. Importación de módulos

Permite usar librerías externas.

import math

print(math.sqrt(16))
